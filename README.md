## 建站用 VPS 还是虚拟主机？先搞清楚这个

## 新手必懂的几个概念（不懂这些会踩大坑）

### 1. KVM 虚拟化 vs OpenVZ

KVM 是目前主流的 VPS 虚拟化技术，每台独享内核，支持安装各种系统，兼容性好。OpenVZ 是老技术，共享内核，很多软件跑不起来。**选建站 VPS，认准 KVM，别碰 OpenVZ。**

搬瓦工目前全系 KVM，这一点不用担心。

### 2. CN2、CN2 GT、CN2 GIA 是什么

这是中国电信国际骨干网的几个档次：

- **163 普通线路**：最便宜，高峰期（晚上 7-11 点）容易拥堵卡顿，适合纯预算优先的场景
- **CN2 GT**：电信去程走 CN2，回程普通，白天好用，晚高峰有时卡
- **CN2 GIA**：电信全程双向走 CN2，最高等级，延迟低，稳定性极好——面向国内访问的建站 VPS，这是最推荐的

对做国内访问为主的网站来说，线路质量直接影响访客体验，**建站 VPS 首选 CN2 GIA 线路。**

### 3. 机房在哪儿、延迟多少

一般来说，离你目标用户越近越好：

- **香港机房**：延迟最低，到国内通常 20-40ms，但价格最贵
- **日本（东京/大阪）机房**：延迟也不错，60-80ms，价格适中
- **美国洛杉矶机房**：延迟相对高（约 150ms），但 CN2 GIA 线路下晚高峰也很稳，性价比最高
- **荷兰机房**：适合欧洲用户或需要备用节点

## 新手建站常踩的几个坑

**坑一：买了最便宜的套餐，晚上网站卡成 PPT**

KVM 入门套餐走的是普通 163 线路，白天没问题，晚高峰拥堵。如果你的网站是面向国内用户，这个感知非常明显。建议预算允许的话，直接上 CN2 GIA-E，不纠结。

**坑二：买了服务器，不知道下一步干嘛**

拿到 VPS 以后，SSH 连上去，跑一行命令安装宝塔面板，就有图形界面了。然后在宝塔里一键安装 LNMP 环境，再一键部署 WordPress，真的不需要懂太多命令行。

**坑三：没买域名就开始建站**

VPS 买了之后还需要一个域名，把域名的 DNS A 记录指向你的 VPS IP，访问者才能通过域名找到你的网站。域名可以去 Namecheap、Cloudflare Registrar 等地方注册。

**坑四：忘记开 SSL 证书**

现在没有 HTTPS 的网站，搜索引擎会降权，浏览器也会显示"不安全"。宝塔面板内置了 Let's Encrypt 免费 SSL 证书申请功能，一键搞定，别忘了。
